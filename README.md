\# AI Resume Analyzer



A full-stack AI-powered Resume Analyzer that evaluates resumes against job descriptions using Transformer-based NLP.



\## Tech Stack



\### Frontend

\- React

\- Tailwind CSS



\### Backend

\- Java 17

\- Spring Boot

\- PostgreSQL



\### AI Service

\- Python

\- FastAPI

\- Transformers

\- Sentence Transformers



\## Architecture



React → Spring Boot → FastAPI → PostgreSQL
